.. py:class:: Client(token, base_url = 'https://tapi.bale.ai/bot', async_mode = None, max_workers = 50)

   A client for interacting with the Bale messenger API.

   :param token: The bot token.
   :type token: str
   :param base_url: The base URL for the API. Defaults to "https://tapi.bale.ai/bot".
   :type base_url: str, optional
   :param async_mode: Force async mode. If None, auto-detected.
   :type async_mode: bool, optional
   :param max_workers: Maximum number of worker threads for handlers. Defaults to 50.
   :type max_workers: int, optional

   :returns: The client instance.
   :rtype: Client
